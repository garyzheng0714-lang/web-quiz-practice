## 1. 设计原则

### 1.1 核心原则

**简洁高效**
- 减少不必要的视觉噪音
- 信息层级清晰，用户快速获取关键信息
- 保持界面的一致性

**专业可信**
- 使用稳定的色彩和字体
- 适当的留白增强呼吸感
- 精致的细节处理

**自然流畅**
- 遵循用户心理模型
- 交互反馈及时
- 过渡动画平滑自然

### 1.2 设计决策流程

```
确定功能 → 明确信息层级 → 选择布局 → 应用设计规范 → 细节优化
```

## 2. 色彩系统

### 2.1 品牌色

```css
/* 主品牌色 */
--lark-primary: #3370ff;
--lark-primary-hover: #2957cc;
--lark-primary-active: #2046a6;
--lark-primary-light: #e1eaff;
--lark-primary-lighter: #f0f4ff;
```

**使用场景**
- 主要操作按钮
- 导航高亮状态
- 重要链接文字
- 图标强调色

**禁止使用**
- 大面积背景（使用主色浅变）
- 装饰性元素
- 错误状态（使用错误色）

### 2.2 功能色

```css
/* 成功 */
--lark-success: #00b856;
--lark-success-bg: #e8ffe8;
--lark-success-bg-light: #f5fff5;

/* 警告 */
--lark-warning: #ff8800;
--lark-warning-bg: #fff5e8;
--lark-warning-bg-light: #fffbe8;

/* 错误 */
--lark-error: #ff4d4f;
--lark-error-bg: #ffe8e8;
--lark-error-bg-light: #fff5f5;

/* 信息 */
--lark-info: #3370ff;
--lark-info-bg: #e1eaff;
--lark-info-bg-light: #f0f4ff;
```

**使用场景**

| 功能色 | 使用场景 | 背景色使用 |
|--------|----------|------------|
| 成功 | 操作成功、状态正常 | `bg-[#e8ffe8]` |
| 警告 | 注意事项、待办提醒 | `bg-[#fff5e8]` |
| 错误 | 操作失败、删除操作 | `bg-[#ffe8e8]` |
| 信息 | 提示信息、说明文字 | `bg-[#e1eaff]` |

### 2.3 中性色

```css
--lark-gray-1: #f7f8fa;  /* 浅背景 */
--lark-gray-2: #e5e6eb;  /* 边框线 */
--lark-gray-3: #c9cdd4;  /* 禁用状态 */
--lark-gray-4: #8f959e;  /* 辅助文字 */
--lark-gray-5: #5f6672;  /* 次要文字 */
--lark-gray-6: #43474e;  /* 正文文字 */
--lark-gray-7: #1f2329;  /* 标题文字 */
```

**使用场景**

| 色值 | 用途 | 示例 |
|------|------|------|
| #f7f8fa | 页面背景、浅色区块 | `bg-[#f7f8fa]` |
| #e5e6eb | 边框线、分割线 | `border-[#e5e6eb]` |
| #c9cdd4 | 禁用状态边框 | `border-[#c9cdd4]` |
| #8f959e | 占位符、辅助信息 | `text-[#8f959e]` |
| #5f6672 | 次要说明文字 | `text-[#5f6672]` |
| #43474e | 正文内容 | `text-[#43474e]` |
| #1f2329 | 标题、重要文字 | `text-[#1f2329]` |

### 2.4 特殊色

```css
/* 价格强调 */
--lark-price: #ff4d4f;

/* 链接色 */
--lark-link: #3370ff;
--lark-link-hover: #2957cc;
```

## 3. 排版系统

### 3.1 字号层级

```css
/* 字号系统 - 基于14px基准 */
--lark-text-h1: 20px;   /* 页面标题 - line-height: 28px */
--lark-text-h2: 18px;   /* 区块标题 - line-height: 26px */
--lark-text-h3: 16px;   /* 卡片标题 - line-height: 24px */
--lark-text-body: 14px; /* 正文 - line-height: 22px */
--lark-text-caption: 12px; /* 辅助文字 - line-height: 20px */
--lark-text-mini: 11px;  /* 小标签 - line-height: 16px */
```

**字重规范**

| 用途 | 字重 | Tailwind类 |
|------|------|------------|
| 页面标题 | SemiBold (600) | `font-semibold` |
| 区块标题 | Medium (500) | `font-medium` |
| 卡片标题 | Medium (500) | `font-medium` |
| 正文 | Regular (400) | `font-normal` |
| 辅助文字 | Regular (400) | `font-normal` |
| 强调文字 | SemiBold (600) | `font-semibold` |

### 3.2 使用示例

```tsx
{/* 页面标题 */}
<h1 className="text-[20px] font-semibold text-[#1f2329] leading-[28px]">
  食品饮料行业榜单
</h1>

{/* 区块标题 */}
<h2 className="text-[18px] font-medium text-[#1f2329] leading-[26px]">
  产品排名
</h2>

{/* 卡片标题 */}
<h3 className="text-[16px] font-medium text-[#1f2329] leading-[24px]">
  产品名称
</h3>

{/* 正文 */}
<p className="text-[14px] text-[#43474e] leading-[22px]">
  正文内容描述
</p>

{/* 辅助文字 */}
<span className="text-[12px] text-[#8f959e] leading-[20px]">
  辅助说明
</span>
```

### 3.3 文字截断

```tsx
{/* 单行截断 */}
<div className="truncate">超长文本自动截断显示省略号</div>

{/* 多行截断（2行） */}
<div className="line-clamp-2">多行文本自动截断，最多显示两行内容</div>

{/* 多行截断（3行） */}
<div className="line-clamp-3">多行文本自动截断，最多显示三行内容</div>
```

## 4. 间距系统

### 4.1 8px基准间距

```css
--lark-space-1: 4px;   /* 极小间距 */
--lark-space-2: 8px;   /* 小间距 */
--lark-space-3: 12px;  /* 标准间距 */
--lark-space-4: 16px;  /* 较大间距 */
--lark-space-5: 20px;  /* 大间距 */
--lark-space-6: 24px;  /* 区块间距 */
--lark-space-8: 32px;  /* 页面级间距 */
--lark-space-10: 40px; /* 大区块间距 */
```

### 4.2 使用场景

| 场景 | 间距值 | Tailwind类 |
|------|--------|------------|
| 图标与文字 | 4px | `gap-1` |
| 按钮内文字与图标 | 6px | `gap-1.5` |
| 标签与元素 | 8px | `gap-2` |
| 卡片内元素 | 12px | `gap-3` |
| 区块内元素 | 16px | `gap-4` |
| 卡片列表间距 | 12px | `space-y-3` |
| 区块间距 | 24px | `space-y-6` |
| 页面区块间距 | 32px | `space-y-8` |
| 页面上下边距 | 40px | `py-10` |

### 4.3 内边距规范

```css
/* 按钮内边距 */
--lark-btn-padding-sm: 6px 12px;  /* 小按钮 */
--lark-btn-padding-md: 7px 16px;  /* 中按钮（默认） */

/* 卡片内边距 */
--lark-card-padding: 16px; /* 标准卡片内边距 */
--lark-card-padding-sm: 12px; /* 小卡片内边距 */
--lark-card-padding-lg: 24px; /* 大卡片内边距 */

/* 表单内边距 */
--lark-input-padding: 7px 12px; /* 输入框内边距 */
```

## 5. 圆角系统

```css
--lark-radius-none: 0;
--lark-radius-sm: 4px;   /* 小圆角 - 标签、徽章 */
--lark-radius-md: 8px;   /* 标准圆角 - 卡片、按钮（默认） */
--lark-radius-lg: 12px;  /* 大圆角 - 对话框、面板 */
--lark-radius-xl: 16px;  /* 超大圆角 - 特殊场景 */
--lark-radius-full: 9999px; /* 完全圆角 - 圆形按钮、头像 */
```

### 5.1 使用场景

| 组件 | 圆角值 | Tailwind类 |
|------|--------|------------|
| 按钮 | 4px | `rounded-[4px]` |
| 输入框 | 4px | `rounded-[4px]` |
| 卡片 | 8px | `rounded-[8px]` |
| 标签 | 4px | `rounded-[4px]` |
| 徽章 | 完全圆角 | `rounded-full` |
| 对话框 | 12px | `rounded-[12px]` |
| 头像 | 完全圆角 | `rounded-full` |

### 5.2 圆角组合规范

```tsx
{/* 卡片 + 标签组合 */}
<Card className="rounded-[8px]">
  <Badge className="rounded-[4px]">标签</Badge>
</Card>

{/* 按钮圆角 */}
<Button className="rounded-[4px]">主要按钮</Button>
<Button variant="outline" className="rounded-[4px]">次级按钮</Button>
```

## 6. 阴影系统

```css
/* 阴影层级 */
--lark-shadow-sm: 0 1px 2px 0 rgba(0, 0, 0, 0.05);
--lark-shadow-base: 0 2px 8px 0 rgba(0, 0, 0, 0.08);
--lark-shadow-md: 0 4px 12px 0 rgba(0, 0, 0, 0.1);
--lark-shadow-lg: 0 8px 24px 0 rgba(0, 0, 0, 0.12);
--lark-shadow-xl: 0 12px 32px 0 rgba(0, 0, 0, 0.15);
```

### 6.1 使用场景

| 场景 | 阴影值 | Tailwind类 |
|------|--------|------------|
| 悬停状态（卡片） | base | `shadow hover:shadow-base` |
| 下拉菜单 | base | `shadow-base` |
| 对话框 | md | `shadow-md` |
| 浮层 | lg | `shadow-lg` |
| 模态框遮罩 | - | 使用半透明背景 |

### 6.2 阴影使用原则

```tsx
{/* 卡片默认无阴影，悬停时显示 */}
<Card className="transition-shadow hover:shadow-base rounded-[8px]">
  {/* 内容 */}
</Card>

{/* 始终有阴影的浮层 */}
<div className="shadow-lg rounded-[12px] p-4 bg-white">
  {/* 浮层内容 */}
</div>

{/* 模态框 */}
<DialogContent className="shadow-md rounded-[12px]">
  {/* 模态框内容 */}
</DialogContent>
```

## 8. 交互与动效

### 8.1 过渡动画

```css
/* 标准过渡 */
transition-all duration-200 ease-in-out

/* 颜色过渡 */
transition-colors duration-200

/* 阴影过渡 */
transition-shadow duration-200

/* 变换过渡 */
transition-transform duration-200
```

### 8.2 悬停效果

```tsx
{/* 卡片悬停 */}
<Card className="transition-all duration-200 hover:shadow-base hover:-translate-y-[2px]">
  {/* 内容 */}
</Card>

{/* 按钮悬停 */}
<Button className="transition-colors duration-200 hover:bg-[#2957cc]">
  按钮
</Button>

{/* 链接悬停 */}
<a className="text-[#3370ff] transition-colors duration-200 hover:text-[#2957cc]">
  链接文字
</a>
```

### 8.3 状态反馈

```tsx
{/* 禁用状态 */}
<Button disabled className="
  bg-[#e5e6eb]
  text-[#c9cdd4]
  cursor-not-allowed
  hover:bg-[#e5e6eb]
">
  禁用按钮
</Button>

{/* 加载状态 */}
<Button disabled className="bg-[#3370ff]">
  <Loader2 className="w-4 h-4 animate-spin mr-2" />
  加载中...
</Button>

{/* 错误状态 */}

  border-[#ff4d4f]

  focus:ring-[#ff4d4f]
" />
<span className="text-[12px] text-[#ff4d4f] mt-1">错误提示信息</span>
```

## 附录：设计决策检查清单

### 设计完成前检查

- [ ] 是否使用了正确的品牌色 (#3370ff)
- [ ] 中性色使用是否正确
- [ ] 字号是否遵循层级规范
- [ ] 间距是否使用8px倍数
- [ ] 圆角是否使用4px/8px标准
- [ ] 按钮尺寸是否符合规范
- [ ] 卡片是否有悬停效果
- [ ] 响应式是否考虑移动端
- [ ] 过渡动画是否平滑自然
- [ ] 错误状态是否明确

### 常见错误

| 错误类型 | 错误示例 | 正确示例 |
|----------|----------|----------|
| 圆角过大 | `rounded-xl` (12px) | `rounded-[8px]` |
| 阴影过重 | `shadow-2xl` | `hover:shadow-base` |
| 字号不统一 | 混用15px、17px | 使用14px、16px |
| 间距随意 | `gap-5`, `gap-7` | 使用`gap-4`, `gap-6` |
| 颜色硬编码 | `bg-blue-500` | 使用`bg-[#3370ff]` |
